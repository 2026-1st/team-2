# Team 2 기계학습기초 프로젝트

아주대학교 인공지능융합학과 2026년 1학기 **기계학습기초** 팀 2 프로젝트 저장소입니다.

## 실행 방법

아직 프로젝트 코드가 추가되지 않았습니다. 추후 데이터 분석/모델링 코드가 정리되면 아래 내용을 업데이트합니다.

1. 필요한 패키지 설치
2. 데이터 준비
3. 전처리 및 학습 스크립트 실행
4. 결과 재현 방법 확인

## 협업 규칙

- 작업 전 `main` 브랜치를 최신 상태로 맞춥니다.
- 기능별 브랜치를 만들고 Pull Request로 병합합니다.
- 커밋 메시지는 변경 목적이 드러나도록 작성합니다.
